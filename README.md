# RGitFlow

A collection of Rake tasks intended to emulate Git Flow in a Ruby context.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rgitflow'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rgitflow
